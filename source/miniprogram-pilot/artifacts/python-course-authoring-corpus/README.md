# Python Course Authoring Corpus

Non-runtime authoring content for 699 Python bilingual lesson candidates.
This directory is excluded from miniprogram packaging via project.config.json.

- lessons/: canonical lesson records (JSON)
- asset-specs/: asset requirement specifications for deferred_asset_required lessons
- corpus-manifest.json: full manifest of 699 records
- term-catalog.json: aggregated bilingual term catalog
- content-job-manifest.json: parallel job division plan
